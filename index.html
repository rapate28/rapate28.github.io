Project 1 - Youtube Views

This C# program analyzes simulated YouTube video data by generating random views and likes for a user-specified number of videos. It employs parallel arrays to store views, likes, and like percentages for each video, facilitating easy data manipulation and analysis. The program calculates key statistics such as the most viewed video, least viewed video, most liked video, and least liked video using dedicated methods. Input validation ensures that the user provides a valid number of videos to analyze, ranging from 1 to 9. Finally, the program displays the calculated statistics in a formatted manner using the console output.

Project 2 - Tea Shop

This program is a simple console application designed for managing orders at a tea shop, offering options to process single orders interactively or process multiple orders from a file. It employs a menu system to guide users through various operations, including selecting tea type and size, and managing order processing. Input validation ensures user inputs remain within valid ranges for tea selection and menu choices. Additionally, the program utilizes constant variables to store tea prices and the sales tax rate, enhancing maintainability and readability. File I/O capabilities enable the program to read orders from a designated file, enhancing flexibility for managing multiple orders efficiently. Overall, the program showcases fundamental concepts such as control flow, input/output operations, error handling, and the potential for transaction management.
